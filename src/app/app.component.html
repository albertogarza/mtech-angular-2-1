Create your first custom component

The point of this exercise is for you to create and use a simple component. Please follow these steps:

  1. Create a new file named example.component.ts.
  2. Declare the ExampleComponent class and export it.
  3. Add the @Component decorator to the class.
  4. Assign the string 'my-example-component' to the selector property inside the @Component decorator.
  5. Assign the string 'Example component works!' to the template template inside the @Component decorator.
  6. Add your new component class name to the declarations array.
  7. Include your component below (line 17 of app.component.html).
  8. Bonus: Move your template to a separate file named example.component.html.
